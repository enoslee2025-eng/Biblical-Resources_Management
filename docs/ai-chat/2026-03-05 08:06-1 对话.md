**Claude**: 完成以下改造：

### 1. 登录二维码改造
- 二维码内容改为 `https://zhuneiqr.com/index.html?page=open-login-resource-website-end&get=...&post=...` 格式
- `get` 参数：获取登录摘要信息的 URL（含 AuidA 占位符）
- `post` 参数：进行登录的 URL（含 AuidA 占位符）
- 轮询间隔改为 3 秒，有效期改为 3 分钟（180 秒）
- 兼容新旧两种后端响应格式

### 2. 导出格式转换器（新建）
- **圣经**：`convertBibleToAppFormat()` — 将内部 `books[].chapters[].verses[]` 转为 APP 格式（catalogs、books 字典、bookIndex、contentIndex、content 含 d 富文本片段数组）
- **注释**：`convertCommentaryToAppFormat()` — 按 `b{书卷号}c{章号}` 组织内容，经节用 `#v;` 格式关联，支持 links 字典
- **词典**：`convertDictionaryToAppFormat()` — 词条数组 `[{ id, t, b }]` 格式，HTML 正文

### 3. 导入二维码改造
- 二维码内容改为 `https://zhuneiqr.com/index.html?page=open-import-resource-end&type=X&get=...` 格式
- type: 0=圣经, 1=注释, 2=词典
- `buildImportQrUrl()` 工具函数生成标准 URL

### 4. 导出页面更新
- JSON 下载按钮改为调用格式转换器后再下载
- 二维码生成改为使用 zhuneiqr.com URL 格式

### 5. API 层新增
- `getDownloadInfo()` — 公开接口，供 APP 扫码后获取资源元数据和下载地址

**修改的文件**:
- `website/src/utils/exportFormat.js`: 新建 — 三种资源类型的格式转换器 + 导入二维码 URL 生成器
- `website/src/views/LoginNew.vue`: 二维码 URL 改为 zhuneiqr.com 格式；轮询 3 秒/有效期 3 分钟
- `website/src/views/BibleExport.vue`: 导入转换器；JSON 下载调用转换器；二维码生成改用新格式
