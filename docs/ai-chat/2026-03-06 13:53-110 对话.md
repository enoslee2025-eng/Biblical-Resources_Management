# 对话 110

**时间**: 2026-03-06 16:02

---

**用户**: 智能格式化预览正确显示 1518 节 50 章，但点击「确认导入」后编辑器显示空白，经文没有展示出来。

**Claude**: `confirmSmartFormat()` 和 `confirmImport()` 在写入经文数据后只调用了 `loadCurrentVerses()`（更新 versesText），但没有刷新富文本编辑器（rawRichContent / richTextContent）。编辑器处于「文本编辑」模式时使用的是 rawRichContent，所以内容显示为空。

修复：在 `confirmSmartFormat()` 和 `confirmImport()` 中，`loadCurrentVerses()` 之后加入 `await nextTick()` + `loadRawRichFromChapter()` / `loadRichTextFromChapter()` 刷新富文本编辑器。

**修改的文件**:
- `website/src/views/BibleEditor.vue`:
  - `confirmSmartFormat()`: 改为 async，导入后刷新富文本编辑器
  - `confirmImport()`: 改为 async，两处 loadCurrentVerses 后都加入富文本刷新
