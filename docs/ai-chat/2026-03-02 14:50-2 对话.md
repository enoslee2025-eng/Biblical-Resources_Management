**用户**: 选择文件夹导入时，无法导入文件

**Claude**: 原因是 `FileList` 是与 input 元素绑定的动态引用，`event.target.value = ''` 清空 input 后 FileList 也会变空，导致文件还没读取就丢失了。修复方案是在清空 input 前先用 `Array.from()` 把文件列表复制为数组。

**修改的文件**:
- `website/src/views/BibleEditor.vue`: `onFolderSelected` 中 `event.target.files` 改为 `Array.from(event.target.files)`
- `website/src/views/CommentaryEditor.vue`: 同上
- `website/src/views/DictionaryEditor.vue`: 同上
