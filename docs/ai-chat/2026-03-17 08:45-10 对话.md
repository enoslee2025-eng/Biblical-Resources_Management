**Claude**: 添加了 .doc (旧版 Word OLE2 格式) 文件的导入支持。由于浏览器端的 mammoth.js 只能解析 .docx 格式，采用后端 Apache POI 方案：

2. 前端 `ImportDialog.vue` 和 `CommentaryImport.vue` 改为先尝试 mammoth 解析，失败后自动调用后端接口解析
3. 新增 API 函数 `parseDocFile()` 用于上传 .doc 文件到后端
4. 8 种语言新增 `import_doc_server_parsing` 翻译键

**修改的文件**:
- `server/pom.xml`: 添加 Apache POI poi-scratchpad 和 quarkus-rest-multipart 依赖
- `server/src/.../resource/DocParseResource.java`: 新增 .doc 文件解析接口
- `website/src/api/resource.js`: 新增 parseDocFile API 函数
- `website/src/components/ImportDialog.vue`: .doc 解析逻辑改为先 mammoth 后调后端
- `website/src/views/CommentaryImport.vue`: 同上
- `website/src/i18n/{zh,en,ja,ko,th,el,he,my}.js`: 新增 import_doc_server_parsing 键
