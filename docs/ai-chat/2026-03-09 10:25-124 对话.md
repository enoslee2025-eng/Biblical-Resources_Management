**用户**: 人名标注后，希望能保存到文本中，在阅览中继续显示。

**Claude**: 实现了人名标注持久化到经文数据并在阅览页面显示：

1. **保存标注到数据**：`saveRawRichToChapter()` 在保存经文时，除了保存纯文本 `text`，还会检查 HTML 中是否包含人名标注（`<span class="person-name">`），如果有则保存为 `markedText` 字段。

2. **加载时使用标注文本**：`loadRawRichFromChapter()` 在人名标注开启时优先使用 `markedText`（已保存的标注文本），没有时才自动标注。

3. **阅览页面显示**：BibleRead.vue 在三种显示模式（经典、段落、大字）中都支持渲染 `markedText`，使用 `v-html` 显示带有人名下划线的文本。

4. **关闭标注时清除**：关闭人名标注开关时，会清除当前章节所有经文的 `markedText` 字段。

**修改的文件**:
- `website/src/views/BibleEditor.vue`: `saveRawRichToChapter()` 新增 markedText 保存逻辑；`loadRawRichFromChapter()` 优先使用 markedText；`togglePersonNameMark()` 关闭时清除 markedText
- `website/src/views/BibleRead.vue`: verses 和 loadedChaptersData 新增 markedText 字段；模板使用 v-html 渲染标注文本；新增 person-name CSS 样式
