# 对话 3

**时间**: 2026-03-02 15:01

---

**用户**: 导入的经文文本自带数字序号（如 "1. 天地..."），和编辑器自动生成的节号重复，显示混乱

**Claude**: 在圣经编辑器的导入逻辑中添加了 `stripLineNumbers()` 智能去序号函数：
- 支持多种序号格式：`1.` / `1、` / `1:` / `1：` / `(1)` / `【1】` 等
- 智能判断：只有超过一半的行都带序号时才去除，避免误删正文中的数字
- 同时应用于"当前章节"和"整卷书"两种导入模式

**修改的文件**:
- `website/src/views/BibleEditor.vue`: 添加 `stripLineNumbers()` 函数，在 `confirmImport()` 中对每行经文进行智能去序号处理
