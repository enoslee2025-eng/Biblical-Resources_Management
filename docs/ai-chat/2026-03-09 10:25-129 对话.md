# 对话 129

**时间**: 2026-03-09 14:28

---

**用户**: 导入和智能排版的规则应该统一，以导入为标准。智能排版可选择本章或本卷书。

**Claude**: 统一了导入和智能排版的解析规则，并为智能排版添加了范围选择：

1. **规则统一**：`smartFormatText()` 不再使用独立的 `parseBibleText()`（bibleVerseParser），改为使用与导入完全相同的解析函数：
   - 本章模式：使用 `splitTextToVerses()`（与导入→本章一致）
   - 本卷书模式：使用 `smartSplitBookChapters()` + `splitTextToVerses()`（与导入→整卷书一致）

2. **范围选择**：在导入对话框底部的智能排版按钮旁新增 radio 按钮组：「本章」和「本卷书」，默认为本章。

3. **名称更新**：「智能格式化」→「智能排版」。

**修改的文件**:
- `website/src/views/BibleEditor.vue`: 新增 `smartFormatScope` 状态；重写 `smartFormatText()` 使用导入的解析函数；更新对话框 UI 添加范围选择
- `website/src/i18n/zh.js`: 更新名称，新增 `be_smart_format_chapter`、`be_smart_format_book`
- `website/src/i18n/en.js`: 新增对应英文翻译
- `website/src/i18n/{ja,ko,th,el,he,my}.js`: 新增对应翻译
