**用户**: 参考 App 圣经展示效果截图，优化智能排版和上传规则，使格式符合 App 中的圣经格式（段落标题、经文结构等）。

**Claude**: 根据 App 截图中的圣经格式，实现了段落标题（section headings）的完整支持：

1. **解析器**：`bibleVerseParser.js` 的 `parseStructuredChapters()` 现在会捕获所有段落标题（如"拿细耳人的条例"），保存到 `chapter.sectionHeadings` 中（键为标题出现在哪节经文之前的节号）。

2. **智能排版**：`smartFormatText()` 和 `confirmSmartFormat()` 保留段落标题信息，导入时写入章节数据。

3. **编辑器显示**：
   - 文本编辑模式（raw）：段落标题渲染为加粗独立段落
   - 逐节编辑模式（inline）：段落标题显示在对应经文行上方
   - 保存时自动检测加粗段落是否为标题并保存回 `sectionHeadings`

4. **阅览页显示**：三种显示模式（经典、段落、大字）都会在对应经文前显示段落标题。

5. **智能排版预览**：预览界面也显示段落标题。

**修改的文件**:
- `website/src/utils/bibleVerseParser.js`: `parseStructuredChapters()` 新增 `sectionHeadings` 收集逻辑
- `website/src/views/BibleEditor.vue`: 新增 `currentChapterHeadings` 计算属性；`loadRawRichFromChapter()` 渲染标题；`saveRawRichToChapter()` 检测并保存标题；inline 编辑模式显示标题；智能排版预览显示标题；新增 CSS 样式
- `website/src/views/BibleRead.vue`: `loadedChaptersData` 新增 `sectionHeadings`；三种显示模式的模板添加标题渲染
