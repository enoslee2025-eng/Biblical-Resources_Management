**用户**: 要求在注释阅读页添加编辑功能，每个段落可以独立编辑，编辑器带富文本编辑功能。

**Claude**: 在 CommentaryRead.vue 中实现了段落内联编辑功能：

1. **编辑按钮**：每个段落右上角悬浮显示"编辑"按钮（鼠标移入时显示）
2. **内联富文本编辑器**：点击编辑按钮后，该段落原位展开为 Tiptap 富文本编辑器，包含：
   - 标题输入框（编辑段落标题）
   - 工具栏：加粗(B)、斜体(I)、下划线(U)、删除线(S)、H2/H3/H4标题、无序列表、有序列表、引用块、高亮、撤销/重做
   - 编辑器内容区（所见即所得编辑）
   - 保存/取消按钮
3. **保存逻辑**：保存时将编辑器 HTML 转回纯文本，更新本地 sections 数组，调用 updateResource API 保存到后端
4. **8语言 i18n**：新增11个编辑相关翻译键
