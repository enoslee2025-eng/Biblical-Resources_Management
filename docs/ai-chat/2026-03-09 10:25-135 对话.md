**用户**: 编辑页面头部要更简约时尚。

**Claude**: 完全重新设计了编辑页面头部，风格简约现代：

1. **一行式头部**：Tab 导航（经文编辑 / 基本信息）+ 版本标签 + 未保存圆点 + 操作链接 + 保存按钮，全部在一行 52px 高的头部栏内
2. **自定义 Tab 导航**：不再使用 Element Plus 的 `el-tabs`，改用自定义 `tab-link` 样式，底部黑色 2px 下划线指示当前 Tab
3. **极简元素**：
   - 版本号用浅灰小标签 `version-tag` 展示
   - 未保存状态用一个橙色小圆点 `dirty-dot` 代替 Tag
   - 「历史版本」「阅览」改为灰色文字链接
   - 保存按钮改为黑底白字圆角按钮
4. **去掉 `el-tabs`**：用 `v-show` + 自定义 Tab 切换内容，更灵活

**修改的文件**:
- `website/src/views/BibleEditor.vue`: 模板重构（自定义头部栏 + v-show 内容切换）；CSS 全新设计（panel-header、tab-nav、save-btn 等）
