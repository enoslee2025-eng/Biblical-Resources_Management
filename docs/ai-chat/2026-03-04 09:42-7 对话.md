**Claude**: 诊断发现 Docker 未安装，因此 MySQL 无法启动，导致后端无法连接。解决方案：
1. 用 Homebrew 安装并启动 MySQL（brew install mysql + brew services start mysql）
2. 创建 bible_dictionary 数据库和用户，执行 init-db.sql 初始化所有表
3. 启动 Quarkus 后端（Java 24 需要加 -Dnet.bytebuddy.experimental=true 参数解决兼容性问题）
4. 后端添加 /public/api/auth/dev-login 接口，供开发环境直接登录获取真实 Token
5. 前端快速进入按钮改为调用 dev-login 接口，获得真实 Token 后所有 API 均可正常使用
