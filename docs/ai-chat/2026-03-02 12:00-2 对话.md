**用户**: 网站打不开；导出按钮点击后白屏；主内圣经APP扫码显示无效二维码；没有管理员登录界面

**Claude**: 排查并修复了三个问题：

1. **导出白屏**：Home.vue 中导出跳转路径是 `/export/${id}`，但路由配置写的是 `/bible/export/:id`，路径不匹配。修复为统一使用 `/export/:id`。

2. **APP扫码无效二维码**：导入二维码之前只放了一个 URL 字符串，APP 不认识。改为与登录二维码相同的 JSON 格式：`{action: "import-resource", url: "...", resourceId: ..., title: "...", type: "..."}`，APP 可以通过 action 字段识别。

3. **后台管理**：npm 依赖未安装、服务未启动。已安装依赖并验证构建成功。需要用户手动启动后台 API 和后台管理前端。

**修改的文件**:
- `website/src/router/index.js`: 导出路由从 `/bible/export/:id` 改为 `/export/:id`
- `website/src/views/BibleExport.vue`: 导入二维码改为 JSON 格式（含 action 字段）
