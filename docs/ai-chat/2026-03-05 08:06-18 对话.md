**用户**: 逐节编辑和文本编辑这两个部分也要具备文档编辑的专业工具栏

**Claude**: 将三种编辑模式都统一使用 RichTextEditor 组件（Office 风格工具栏）：
- **逐节编辑**：点击某节经文时，原来的 textarea 替换为 RichTextEditor，支持加粗/斜体/颜色/字体等
- **文本编辑**：原来的纯文本 textarea 替换为 RichTextEditor，每段落 = 一节经文
- 新增数据同步逻辑：inlineRichContent（逐节模式）、rawRichContent（文本模式）
- 更新了保存、自动保存、切换书卷/章节/模式时的数据同步

**修改的文件**:
- `website/src/views/BibleEditor.vue`:
  - 新增 `inlineRichContent`、`rawRichContent` 等响应式变量
  - 新增 `loadRawRichFromChapter()`、`saveRawRichToChapter()`、`saveInlineRichToVerse()` 函数
  - 模板：逐节模式 textarea → RichTextEditor，文本模式 textarea → RichTextEditor
  - 移除旧的 font-toolbar（字号/行距/字体选择器）
  - 更新 `handleSave`、`handleAutoSave`、`onSelectBook`、`onSelectChapter`、模式切换 watch
  - 清理 `.iv-textarea` 等不再使用的 CSS
