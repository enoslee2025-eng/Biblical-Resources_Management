# 对话 9

**时间**: 2026-02-28 16:06

---

**用户**: 点击退出登录没有回到主界面；注释和词典没有批量导入功能

**Claude**: 修复了两个问题：

1. **退出登录不跳转**：原因是 `logout()` API 失败时被 catch 当作用户取消处理。修复为把确认弹窗和退出逻辑的 try-catch 分开，即使 API 失败也会跳转。

2. **注释/词典缺少批量导入**：为注释和词典编辑器都添加了批量导入弹窗，支持粘贴文本和上传TXT文件。注释格式为经文引用+注释内容，词典格式为词条+释义，用空行分隔。

**修改的文件**:
- `website/src/views/Home.vue`: 修复退出登录逻辑
- `website/src/views/CommentaryEditor.vue`: 添加批量导入功能
- `website/src/views/DictionaryEditor.vue`: 添加批量导入功能
- `website/src/i18n/zh.js`: 新增导入格式提示翻译
- `website/src/i18n/en.js`: 新增导入格式提示翻译
